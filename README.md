# 英語・インドネシア語 ASR分別＆逆再生実験プロジェクト

## 1. 実験の目的・仮説
- **目的:** 15単語程度の脈絡のない会話（例：「おはよう（英）」→「犬はどこだ（印）」→「パンケーキ食べよう（英）」）が含まれる音声から、英語とインドネシア語の区間を正しく分別・認識する。
- **実験要素（逆再生）:** 音声を逆再生（時間反転）させたとき、文脈や単語の意味が破壊されても、AIが「音響特徴（周波数の質感やイントネーションの波）」だけで言語を判別できるかを検証する。

## 2. 実験の構成案（アーキテクチャ）
システムは「音声を画像に変換して、画像認識AIに解かせる」というアプローチを採用する。

1. **データ準備:** - Mozilla Common Voice等のオープンデータから、英語とインドネシア語の短文・挨拶音声をダウンロード。
   - プログラムでこれらをランダムに結合し、「脈絡のない複数言語の会話音声」を大量に自動合成する（データ拡張）。
2. **特徴量抽出:**
   - 音声信号を「Log-Mel Spectrogram（対数メルスペクトログラム）」に変換し、縦軸＝周波数、横軸＝時間、画素値＝音の強さ、の【2次元画像】にする。
3. **モデル:**
   - 画像認識の王道モデル「ResNet（既製品）」を使用。出力層を「英語/インドネシア語」の2クラス分類用にカスタマイズする。

## 3. システム開発スタック
- **フロントエンド:** React（音声アップロード、逆再生トグル、結果のグラフ可視化）
- **バックエンド:** PHP（リクエスト中継、実験ログのDB保存など）
- **AI推論サーバー:** FastAPI / Python（音声をメルスペクトログラム画像に変換し、PyTorchで推論）

## 4. 最初期の実験ロードマップ
- **Step 1:** FastAPI単体で、音声を読み込んでメルスペクトログラム（画像）に変換して表示するテスト（視覚的確認）。
- **Step 2:** 順再生の音声画像を使って、AIに言語判別を学習させ、精度（ベースライン）を確認する。
- **Step 3:** 音声画像を左右反転（逆再生）させてAIに入力し、順再生のときと比べてどれくらい精度が維持できるかを比較実験する。
